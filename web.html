Restaurant Management System

The Restaurant Management System is a Java-based application designed to handle essential functions within a restaurant setting, including menu management, ordering, billing, customer tracking, and staff management. This system is organized using object-oriented principles, creating specific classes to represent real-world entities such as Menu items, Customers, Bills, and Staff (e.g., Waiters). The project involves using both single and doubly linked lists to store and manage dynamically linked data, allowing for efficient manipulation of the restaurant's information.
